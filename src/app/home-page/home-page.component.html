Carrie Beaton's Brand Marketing Code Test

This page was created using Bootstrap and NG Bootstrap for responsive design. NGRX was used to create a unidirectional data flow and avoid mutating state. State management is a great way to help keep an application maintainable and scalable and it is easily testible. Reactive forms were used because they are easier to scale and there is less setup for testing. Unit testing was added throughout to demostrate understanding. I very much enjoyed this project and look forward to a code review!
